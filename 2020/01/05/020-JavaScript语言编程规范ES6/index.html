020-JavaScript语言编程规范ES6

发表于 2020-01-05

JavaScript语言编程规范ES6

代码风格

  • 导出的默认函数使用驼峰命名、文件名与函数名一致；
  • 导出单例、函数库、空对象时使用帕斯卡式命名，即首字母大写，如DisplayInfo；

类型和变量

  1. 变量必须显式声明作用域
    • var 声明全局变量或函数级变量；
    • let 声明块级局部变量；
    • const 声明块级的只读作用域；（变量值可修改，重新赋值会出错）
  2. 尽量对所有引用使用const，const和let的作用域更小；
  3. 将所有的const和let分组；
  4. 在需要的地方给变量赋值，但必须放在合理的位置；

对象和引用

1. 创建有动态属性名的对象时，尽量在一个地方定义其所有属性；
 2. 使用对象方法的简写，即`const a=a`，const obj={a,};

数组

  1. 使用扩展运算符…复制数组

    const itemsCopy=[…items]

属性

使用. 来方问对象属性，只有属性的动态属性时才使用[]

逗号、分号

  • 逗号写在行尾，且增加结尾的逗号。
  • 使用分号，以分号作为语句结束符。

函数

  1. 使用函数声明代替函数表达式；
  2. 不要使用arguments，可以选择rest语法…替代
3. 直接给函数的参数指定默认值，不使用变化的函数参数。
 4. 直接给函数参数赋值时，要避免副作用。

代码块

1. 使用括号包裹所有的多行代码块。
 2. 使用if、else多行代码块，把else放在代码块关闭括号的同一行。

模块

  1. 代码总是使用es6标准模块import/export方式，而不使用非标准的模块加载器。

  2. 不要使用* 的import。

  3. 不要从import中直接export

    import {es6} from ‘./ablier.js’;

    export default es6;

  4. 如果文件只输出一个类，那文件名必须和类名完全保持一致。

箭头函数

  1. 不要保存this的引用，使用箭头函数或Function.bind。

  2. 当必须使用函数表太式时，使用箭头函数；

    [1,2,3].map((x)=>{

    ​ return x*x;

    })

  3. 如果一个函数适用一行写出并且只有一个参数，那就把花括号、圆括号和return都省略掉

    [1,2,3].map(x => x*x;);

构造器

  1. 使用class关键字定义类
  2. 采用extends关键字实现继承

解构

  1. 使用解构存取和使用多属性对象，“es6允许按照一定的模式，从数组对象中提取值、对变量赋值，这称之为解构”
  1. 将数组成员赋值给变量时，使用数组解析；
  2. 需要回传多个值时，使用对象解构，而不是数组解构；

String

  1. 长度超过80的字符串应该使用字符串连接换行；
  2. 构建字符串时，使用字符串模板而不是字符串连接；
  1. 数组遍历采用for/of，对象遍历采用for/in

存取器

  1. 属性的存取函数不是必须的，如果需要存储函数使用get、set方法，如果属性是布尔值，存取函数是isVal()或hasVal()
  1. 创建get(),set()函数要保持一致
